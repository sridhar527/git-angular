git init  ->   intialisation
git add README.md ->   add to index 
git commit -m "first commit"   -> add to local repository
git remote add origin https://github.com/sridhar527/git-angular.git  -> added remote repository 
git push -u origin  -> add code to remote repository

git checkout <feature_branch>
